# **App Name**: Market Snapper

## Core Features:

- Symbol Input: Symbol Input: Allow users to input a stock or crypto symbol.
- Timeframe Selection: Timeframe Selection: Users select from a specific timeframe ('1 hour' or '3 min') to pull the data.
- Data Fetching: Data Retrieval Tool: Fetch the latest 5 records for the provided symbol and timeframe from dummy data. Dynamically decide whether to include the ohlcv values in the result.
- Data Display: Display Data: Present the OHLCV (Open, High, Low, Close, Volume) data in a clean table or chart format.
- Error Handling: Error Handling: Gracefully handle invalid symbols or data retrieval issues.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust and stability.
- Background color: Very light blue (#E8EAF6), almost white, for a clean interface.
- Accent color: Orange (#FF9800) for interactive elements and important data points, such as CTAs.
- Font pairing: Use 'Space Grotesk' (sans-serif) for headers and 'Inter' (sans-serif) for body text, providing a clean and modern feel.
- Simple, geometric icons for representing OHLCV data or related actions.
- Clear, tabular layout for data presentation; intuitive input fields for symbol and timeframe selection.
- Subtle transition effects for data updates and interactive feedback.
